# Leprechaun Name Generator

A Flask-based web application that generates creative leprechaun names and provides downloadable PDF certificates.

## Features

- **Name Generation**: Generate unique leprechaun names with descriptions
- **PDF Downloads**: Create professional PDF certificates with custom names
- **Responsive Design**: Mobile-friendly interface
- **Security Features**: Rate limiting, input validation, secure headers
- **Monetization Ready**: Freemium model with watermarking

## Installation

```bash
# Clone repository
git clone https://github.com/Masteringh/leprechaun-generator.git
cd leprechaun-generator

# Install dependencies
pip install -r requirements.txt

# Run application
python app_secure.py
```

## Requirements

- Python 3.9+
- Flask 3.0.0
- WeasyPrint 61.0
- Flask-Limiter 3.5.0

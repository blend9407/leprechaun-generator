# PDF Template Designs - Leprechaun Name Certificates

## Template 1: Classic Emerald Certificate

### Design Elements:
- **Background**: Parchment texture with subtle Celtic knot pattern
- **Border**: Intricate Celtic knot border in emerald green
- **Colors**: Emerald green (#2E8B57), Gold (#FFD700), Dark green (#006400)
- **Typography**:
  - Title: 'Luckiest Guy', 48pt, Gold with green shadow
  - Name: 'Irish Grover', 60pt, Dark green
  - Details: 'Open Sans', 14pt, Charcoal
- **Graphics**:
  - Top center: Leprechaun hat icon
  - Corners: Shamrock motifs
  - Bottom: Pot of gold illustration
- **Layout**:
  - Certificate size: A4 (210mm × 297mm)
  - Margins: 25mm all sides
  - Content centered vertically and horizontally

### Content Sections:
1. Header: "Official Leprechaun Name Certificate"
2. Generated name (large, prominent)
3. Meaning/description of the name
4. Decorative separator line
5. Issue date and certificate number
6. Footer: "This certifies that [Real Name] possesses the magical qualities of a true leprechaun"
